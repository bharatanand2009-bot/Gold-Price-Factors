# Gold-Price-Factors
Identifying the qualitative and quantitative factors most strongly associated with gold prices in India.

README###
This project investigates factors affecting the gold prices in India between 2017 and 2025. It uses monthly economic and geopolitical data, alongside linear regression analysis conducted in Microsoft Excel to identify the factor with the highest correlation to the price of gold. Qualitative and Quantitative variables were used including: Geopolitical Events, Commodity inflation shocks, inflation FX Events, cultural significance, inflation, interest rates, India VIX and Forex Rate (USD to INR). Results are summarized in the accompanying research poster.
